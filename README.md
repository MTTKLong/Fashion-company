# Fashion Company - Project Skeleton
React + Tailwind frontend and PHP (no framework) backend (API + Tabler admin skeleton).

This skeleton provides minimal starter files to save you setup time. It is **not** a finished product,
but includes working examples and essential configs to get development started.

## Contents
- frontend/ (Vite + React + Tailwind)
- backend/ (PHP API endpoints + admin folder skeleton)
- sql/fashion_company.sql (initial schema)
- INSTALL.md (how to run locally)
